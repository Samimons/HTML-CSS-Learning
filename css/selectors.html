Descendant combinator

The " " (space) combinator selects elements that are descendants of the first element.

Child (direct) combinator

The > combinator selects elements that are direct children of the first element.

Compound selector

We can combine class, id, and element selectors (without spaces). This example selects divs with the class "special".
