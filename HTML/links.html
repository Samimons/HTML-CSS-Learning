Hyperlinks

HTML links are hyperlinks.

You can click on a link and jump to another document locally or a completely different URL.

The HTML <a> tag defines a hyperlink. It has the following syntax:

Example

<a href="https://www.google.com/">link to Google</a>

Link to Google

You can have as many attributes as you want, just make sure each one of them is separated with a space, and they all go into the opening tag.

<tag attribute="value" anotherattribute="value">Content</tag>

The target Attribute

By default, the linked page will be displayed in the current browser window. To change this, you must specify another target for the link.

The target attribute specifies where to open the linked document.

The target attribute can have one of the following values:

  • _self - Default. Opens the document in the same window/tab as it was clicked
  • _blank - Opens the document in a new window or tab
  • _parent - Opens the document in the parent frame
  • _top - Opens the document in the full body of the window

Absolute URLs vs. Relative URLs

The example above is using an absolute URL (a full web address) in the href attribute.

A local link (a link to a page within the same website) is specified with a relative URL (without the "https://www" part):

Example

  <a href="index.html">Index Page</a>

Index Page

Use an Image as a Link

To use an image as a link, just put the <img> tag inside the <a> tag:

Example

  <a href="https://www.google.com"><img src="../images/black puppy.jpg" 
 alt="Black puppy" style="width: 45px; height: 45px;"></a>


Link Titles

The title attribute specifies extra information about an element. The information is most often shown as a tooltip text when the mouse moves over the element.

Example

    <a href="https://www.google.com" title="A link to Google.com">Google.com</a>

Google.com

Bookmark Link

Bookmarks can be useful if a web page is very long.

To create a bookmark - first create the bookmark, then add a link to it.

When the link is clicked, the page will scroll down or up to the location with the bookmark.

First, use the id attribute to create a bookmark:

Example

<h2 id="C4">Chapter 4</h2>

Then, add a link to the bookmark ("Jump to Chapter 4"), from within the same page:

Example

<a href="#C4">Jump to Chapter 4</a>

You can also add a link to a bookmark on another page:

Example

<a href="html_demo.html#C4">Jump to Chapter 4</a>
